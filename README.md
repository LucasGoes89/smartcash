# 💰 SmartCash

Dashboard financeiro moderno desenvolvido com HTML, CSS e JavaScript.

## 🚀 Funcionalidades

- Cadastro de receitas e despesas
- Cálculo automático de saldo
- Persistência com LocalStorage
- Layout responsivo

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript
